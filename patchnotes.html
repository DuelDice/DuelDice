Duel Dice
Patch Notes







Version 1.05
2/28/2018


USER INTERFACE

-Added UI Feedback for when your opponent purchases a Demon Pact
-Added a "Return to Lobby" button when searching for a game
-Added an Options menu that contains Volume control
-Updated Icons with white backgrounds for easy readability on mobile
-Added the username of the Demon currently in use to the Start Screen
-Moved Leaderboard and Credits buttons on Start Screen
-Updated Credits Screen
-Made tutorial button bigger in lobby and moved it under Credits
-Edited colors and positions of buttons in FateShop
-Leaderboards Screen uses toggle in the center
-Made Hover Tooltips more readable
-Usernames now hover in-game when it is that demon's turn


DICE

-Changing Dice Faces now cost 20 per face
-Buying extra dice now cost 100 per dice
-Free re-rolls no longer scale with the number of Dice purchased. Now all players will only have 3 free re-rolls during their off turn


BUG FIXES

-Fixed instances of the word "Flag" showing up as "Scale"
-Fixed Android devices setting turn timer to Zero on sleep
-Fixed some networking connectivity issues
